# Philosophy: Anti-Lock-In, Health-First, Autonomy-Preserving OS

This OS exists to free humans from digital captivity, not contribute to it.
- Interoperability by default (driver packages, no platform lock-in)
- Govern-before-generate (no action without gates + kernel OPEN state)
- Deterministic audit (signed receipts + /replay match:true)
- Healthy variation by default (no addictive loops; encourage outdoor/natural rhythms)
- Privacy-first (mask faces/plates by default; fuzzy geo unless whitelisted)
- User sovereignty (ASK/EXPLAIN unless OPEN; consent remembered per session)
